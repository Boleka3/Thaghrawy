          How to use Thaghrawy

          Thaghrawy is an AI-assisted penetration-testing harness with persistent, cross-engagement memory. You drive it against your own authorized targets; the agent runs ~40 security tools, and you stay in the loop.

          1. Engagements (left panel)

          • Create — enter a name + target, pick a mode, hit [+] NEW. Recon Only withholds the exploitation tools entirely.
          • Select — click an engagement's name. Each engagement has its own chat, findings, and reports; switching swaps the whole workspace.
          • Edit / Delete — the edit and del buttons on each row rename/retarget or remove it (delete also removes its findings).
          • Creating an engagement for a target that already exists reuses that engagement, so work for one target stays together.

          2. Workflow (phase banner)

          Enumeration → Collaboration → Reporting. Hit Enumerate to run an autonomous recon sweep that auto-ingests easy findings, then hands off to you. In collaboration, every tool call the model proposes waits for your approval.

          3. Chatting with the agent

          Type in the bottom input to instruct the agent. It streams its reasoning, proposes tool calls, and (in collaboration) pauses for approve / reject / edit. Use /auto safe or /auto all to stop clicking; /auto off to resume gating.

          4. Running a tool yourself

          Tools lists every tool (click a name to fill the form). Run tool opens a form: tool name + JSON args (e.g. {"target":"http://juice-shop:3000"}). The result is fed back to the model.

          5. Interrupt & Compact

          • Interrupt — halt the current turn and send a new instruction in one go. (Stop just halts.)
          • Compact — summarize the session and shrink the context. Use it when a long turn gets slow or the model starts erroring on a bloated context.

          6. Findings & reports (right panel)

          Findings collect on the right. Click a finding to see its full details. On any tool result, →finding promotes it into a reviewable finding; edit curates it; FP marks a false positive. Report (or [+] Generate Reports) builds a technical and an executive report you can download.

          Slash commands

          /help /tools /run <tool> {json} /enumerate /suggest /draft /promote /approve /reject /edit /auto off|safe|all /phase collab|report /report /compact /interrupt <text> /stop

          Keyboard shortcuts

          • Ctrl+` — focus chat input
          • Ctrl+Enter — send message
          • Escape — close modal / toast
